# 无限剪贴板

一个轻量的 Windows 剪贴板历史工具：常驻后台，自动记录剪贴板，支持搜索、恢复、收藏、自动过期清理、保留时间设置和开机自启动监控。

## 启动

双击 `启动无限剪贴板.vbs`。这个入口不会弹出 cmd 黑框。

`启动无限剪贴板.bat` 保留为备用入口。

## 分发给其他人

运行 `Build-Installer.ps1` 后，会在 `dist` 文件夹生成 1.0.2 版单文件原生安装包 `InfiniteClipboard-Setup-1.0.2.exe`，同时自动更新 `website/download` 中供下载页使用的文件。接收方双击后进入原生安装向导：默认安装到当前用户的软件目录，也可自定义安装位置；程序本体、安装和卸载均为原生 Windows EXE，不会启动 PowerShell、CMD 或 VBS 窗口。安装不需要管理员权限，会创建桌面和开始菜单入口，并在用户点击“完成”关闭安装向导后按勾选项启动程序。可通过软件设置、Windows 的“已安装的应用”，或开始菜单中的“卸载无限剪贴板”卸载；为保护历史记录，卸载时默认保留 `%APPDATA%\InfiniteClipboard` 中的数据。

## 下载网站

`website/index.html` 是可静态部署的下载介绍页。将整个 `website` 文件夹上传到任意静态网站托管服务后，访问首页即可下载安装包；每次重新运行构建脚本，下载文件会自动同步到该文件夹。

## 软件更新通道

构建会同时生成 `website/releases.json`，其中包含最新版本、安装包相对地址与 SHA-256 校验值。发布网站时应一并上传该文件。

首次发布到自己的 HTTPS 地址后，构建安装包时传入发布清单地址，例如：

`./Build-Installer.ps1 -UpdateFeedUrl 'https://你的域名/releases.json'`

这样安装后的客户端会默认开启“自动更新软件版本”：后台检查新版本、在“设置”按钮上显示红点，并在设置页提供“最新版本 / 当前版本 / 更新本软件”。下载完成且校验通过后，程序会退出、安装新版本并自动重启。也可在设置页调整更新发布地址。

程序已做单实例保护：重复启动不会再创建新的托盘图标，而是唤出已经运行的窗口。

新版首次运行会默认开启「开机自启动后台监控」。启动项使用无黑框的后台入口；以后点右上角关闭按钮只会隐藏页面到系统托盘，剪贴板监听不会停止。程序会依次尝试 `Ctrl + 1` 到 `Ctrl + 0`，自动使用第一个未被占用的组合键；也可双击托盘图标或在托盘菜单选择「打开」。只有托盘菜单里的「退出」才会完全结束程序。

如果升级前已经开出了多个旧后台，可以先运行 `关闭重复后台.bat`，再重新启动一次新版。

## 常用操作

- `Ctrl + 数字`：默认从 1、2……9、0 中自动分配第一个可用组合；也可在设置页点击快捷键输入框后直接按下新组合。软件会先检测系统级冲突，成功后才保存并启用。
- 双击历史项、点最右侧「复制」列的 ⧉ 图标，或选中后按 `Enter`：恢复到系统剪贴板。
- `Delete`：删除选中记录。
- `Ctrl + F`：聚焦搜索框。
- 历史列表「类型」位置的下拉框可以筛选全部、文本、图片、文件或收藏。
- 「收藏/取消」：收藏项不会被保留时间自动清理。
- 复制成功后，窗口中间会短暂显示「复制成功」。

## 已支持的剪贴板类型

- 文本
- 图片（保存 Windows 剪贴板里的 DIB 数据，可恢复粘贴）
- 文件列表（复制文件时的路径列表，可恢复到剪贴板）

## 文件保存设置

点顶部「设置」可以调整：

- 单条保存上限：默认 1MB。
- 记录类型：文本、图片、文件，可多选。
- 保留时间：24 小时、48 小时、一周、两周、一月、三月、一年、永久。
- 多文件归档：默认关闭。关闭时，复制多个文件/文件夹不会写入历史；开启后会在数据目录里创建独立文件夹保存副本，并恢复这些副本路径到剪贴板。
- 暂停记录剪贴板。
- 开机自启动后台监控。
- 记录新内容时系统通知。
- 界面字号：小、标准（默认）、大、特大四档。
- 快捷键输入框、冲突检测与“重新分配”；重新分配会恢复为自动选择首个可用的 Ctrl + 数字组合。
- 文件列表列宽可拖动调整；窗口宽度会同步变化并在退出后保留。筛选栏可查看当前快捷键，也可一键恢复默认列宽。
- 自动检查更新、更新发布地址与手动安装更新。

### 自动清理与通知

- 记录新内容时系统通知：每次成功保存一条剪贴板历史时，弹出一个 Windows 通知。默认关闭，因为高频复制时会比较吵。
- 自动过期清理：程序后台会定时删除已经超过保留时间、且未收藏的历史项。

## 保留策略

内置档位：

- 24 小时
- 48 小时
- 一周
- 两周
- 一月
- 三月
- 一年
- 永久

数据默认保存在：

`%APPDATA%\InfiniteClipboard`

## 设计取舍

- 不限制历史条数；主要通过保留时间自动清理。
- 单条记录默认最大 1MB，避免误复制超大图片或大目录时拖慢系统。
- 重复内容会自动去重并移动到最新位置。
- 收藏项默认视为长期重要内容，不参与过期清理。
